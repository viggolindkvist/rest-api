Documentation

Routes

  • GET /
    Home
  • POST /user
    Create a new user
  • POST /login
    Login to an existing account
  • 🔐 GET /users
    Show all users. Token required
  • 🔐 GET /users/me
    Shows information from current token
  • 🔐 GET /users/{identifier}
    Specify an identifier (id or username) to get corresponding user
  • 🔐 PUT /users/{id}
    Updates a specific users credentials
  • Routes marked with 🔐 requires a valid token
