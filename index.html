Routes

    Login to an existing account
  • 🔐 GET /users
    Show all users. Token required
  • 🔐 GET /users/{specifier}
    Specify an id to get corresponding user. /users/me shows current token user
  • 🔐 PUT /users/{id}
    Updates a specific users credentials
  • Routes marked with 🔐 requires a valid token
